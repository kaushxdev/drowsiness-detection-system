# Real-Time Driver Drowsiness Detection System

Detects driver drowsiness in real time using computer vision and alerts the user.

## 🛠 Tech Stack
- Python
- OpenCV
- Dlib / Mediapipe
- NumPy

## 🚀 Features
- Real-time webcam monitoring
- Eye aspect ratio detection
- Drowsiness alert system
- Live dashboard

## ▶️ How to Run
pip install -r requirements.txt  
python app.py
